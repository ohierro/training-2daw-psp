# Sesión 2: JavaScript Moderno y TypeScript

## Objetivos de Aprendizaje

Profundizar en JavaScript moderno y adquirir competencias en TypeScript para código más robusto y mantenible.

## Contenidos

### Asincronía avanzada

Repaso de *Promises* y el patrón `async/await` para evitar el "callback hell".

- Callbacks y sus limitaciones
- Introducción a Promises
- Encadenamiento con `.then()`, `.catch()`, `.finally()`
- El patrón `async/await`
- Manejo de errores en código asincrónico
- Promise.all(), Promise.race(), Promise.allSettled()

### Fundamentos de TypeScript

Tipado estático, tipos personalizados (`type`) e interfaces.

- ¿Por qué TypeScript?
- Sistema de tipos básicos
- Tipos personalizados con `type`
- Interfaces vs tipos
- Genéricos
- Uniones e intersecciones de tipos

### Clases y Decoradores

Introducción a la sintaxis de clases moderna y el concepto de **Decoradores** (explicar que son funciones que modifican el comportamiento de una clase o método).

- Sintaxis de clases en TypeScript
- Modificadores de acceso (`public`, `private`, `protected`)
- Constructores y propiedades
- Métodos y getters/setters
- Conceptos de decoradores
- Decoradores de clase
- Decoradores de método y propiedad

### Configuración

El archivo `tsconfig.json` y el proceso de transpilación.

- Estructura del `tsconfig.json`
- Opciones de compilación importantes
- Targeting y módulos
- Strict mode
- Compilación a JavaScript
- Integración con herramientas de construcción
