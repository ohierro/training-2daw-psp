# 2EV - Curriculum Node.js/NestJS

## Contenido del Curso

Este curso está organizado en 8 sesiones que cubren desde los fundamentos de Node.js hasta la persistencia de datos con TypeORM.

### Sesiones del Curso

1. **[Sesión 1: El Ecosistema Node.js](./sesion-01-ecosistema-nodejs/index.md)**
   - Introducción al motor V8 y el Event Loop
   - Gestión de paquetes con NPM
   - Módulos en Node.js
   - Globales y utilidades

2. **[Sesión 2: JavaScript Moderno y TypeScript](./sesion-02-javascript-typescript/index.md)**
   - Asincronía avanzada
   - Fundamentos de TypeScript
   - Clases y Decoradores
   - Configuración

3. **[Sesión 3: Fundamentos de Express](./sesion-03-fundamentos-express/index.md)**
   - Servidor HTTP básico
   - Sistema de Rutas (Routing)
   - Middlewares
   - Métodos HTTP

4. **[Sesión 4: Hola, NestJS: Arquitectura](./sesion-04-nestjs-arquitectura/index.md)**
   - Nest CLI
   - Estructura de archivos
   - Arquitectura modular
   - Arranque de la aplicación

5. **[Sesión 5: Controladores y Rutas en NestJS](./sesion-05-controladores-rutas/index.md)**
   - Decoradores de clase
   - Decoradores de método
   - Manejo de la petición
   - Respuestas HTTP

6. **[Sesión 6: Proveedores y Servicios (Inyección de Dependencias)](./sesion-06-proveedores-servicios/index.md)**
   - El decorador @Injectable
   - Inyección de Dependencias (DI)
   - Singletons
   - Separación de responsabilidades

7. **[Sesión 7: Validación de Datos (DTOs) y Pipes](./sesion-07-validacion-datos/index.md)**
   - Data Transfer Objects (DTO)
   - Validación automática
   - ValidationPipe
   - Transformación de datos

8. **[Sesión 8: Persistencia de Datos Inicial (TypeORM)](./sesion-08-persistencia-datos/index.md)**
   - Configuración de TypeORM
   - Entidades
   - Repositorios
   - Cierre del flujo

---

### Recomendación para la Sesión 4

Para la **Sesión 4**, se recomienda mostrar este video que explica visualmente la arquitectura de NestJS, lo cual les ayudará a entender por qué usamos módulos y servicios antes de empezar a picar código.

**[Curso de NestJS para principiantes](https://www.youtube.com/watch?v=1hpc70_OoAg)**

Este video es muy útil porque cubre desde la instalación de Node.js hasta la creación de rutas, ideal para reforzar los conceptos de las primeras sesiones de tu calendario.
