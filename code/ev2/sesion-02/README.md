# Proyecto de ejemplos JavaScript y TypeScript

Este proyecto contiene ejemplos extraídos de las slides de la sesión 02.

## Estructura
- js/: Ejemplos JavaScript
- ts/: Ejemplos TypeScript
- tsconfig.json: Configuración básica para TypeScript

## Cómo usar
- Instala dependencias: `npm install`
- Compila TypeScript: `npx tsc`
- Ejecuta ejemplos JS: `node js/<archivo>.js`
- Ejecuta ejemplos TS: `node dist/<archivo>.js` (tras compilar)
