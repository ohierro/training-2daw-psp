# Resumen de Contexto para IA

Este repositorio está organizado para facilitar el trabajo con IA y generación de materiales educativos:

- **code/**: Ejemplos de código en Java, JavaScript y TypeScript. Cada subdirectorio representa una unidad o tema.
- **presentations/**: Slides en Markdown, compatibles con Marple para generar HTML.
- **public/**: Carpeta de salida para publicar en GitHub Pages (gh-pages).
- **scripts/**: Scripts para generar los HTMLs, incluyendo una portada navegable.

Usa este resumen para orientar a la IA sobre la estructura y propósito de cada carpeta.
